## 로그인

1. 사이트에 접속하면 랜딩 페이지가 나타납니다.
2. 우측 상단 **팀 로그인** 또는 사이드바 진입 시 로그인 페이지로 이동합니다.
3. 등록된 이메일과 비밀번호를 입력하고 **로그인** 버튼을 누르세요.
4. 로그인 성공 후 대시보드로 자동 이동합니다.

## 대시보드

로그인 후 가장 먼저 보이는 화면입니다.

| 영역 | 설명 |
|------|------|
| **KPI 카드 (상단)** | 전체 아이디어 수, 승인/구현 완료, 논의 중, 활성 프로젝트 수를 실시간으로 보여줍니다. |
| **사업부별 현황** | HeungHan / Substract Lab / Sensus 각 사업부의 아이디어·프로젝트 수를 확인할 수 있습니다. 카드를 클릭하면 해당 사업부 페이지로 이동합니다. |
| **최근 아이디어** | 가장 최근 등록된 아이디어 5개가 표시됩니다. |
| **팀 활동 내역** | 아이디어 등록, 댓글, 상태 변경 등 팀원들의 최근 활동을 볼 수 있습니다. |

## 아이디어 보드

팀의 모든 아이디어를 모으고, 논의하고, 발전시키는 공간입니다.

### 뷰 모드 (우측 상단 토글)

| 아이콘 | 모드 | 설명 |
|--------|------|------|
| 격자 | **그리드 뷰** | 카드 형태로 아이디어를 한눈에 봅니다. |
| 보드 | **칸반 보드** | 제안 → 논의중 → 승인 → 구현완료 열로 드래그하여 상태를 변경합니다. |
| 달력 | **타임라인** | 시작일/종료일이 설정된 아이디어를 간트 차트로 봅니다. |

### 아이디어 등록

1. **새 아이디어** 버튼 클릭
2. 프로젝트 선택 → 제목 입력 → 에디터로 내용 작성
3. 우선순위와 태그 설정 후 **아이디어 추가**

### 에디터 기능

리치 텍스트 에디터로 다양한 서식을 사용할 수 있습니다:

- **텍스트 서식**: 굵게, 기울임, 밑줄, 취소선, 형광펜, 인라인 코드
- **제목**: H1 / H2 / H3
- **리스트**: 글머리 기호, 번호 매기기, 체크리스트
- **정렬**: 왼쪽 / 가운데 / 오른쪽
- **블록**: 인용문, 코드 블록, 구분선
- **삽입**: 링크, 이미지(URL), 표(3x3)
- **실행 취소 / 다시 실행**

### 아이디어 카드 메뉴 (점 세 개 아이콘)

- **발전시키기**: 기존 아이디어를 기반으로 새 아이디어를 생성합니다.
- **수정**: 제목, 내용, 우선순위, 태그를 수정합니다.
- **링크 복사**: 아이디어 링크를 클립보드에 복사합니다.
- **상태 변경**: 제안 / 논의중 / 승인 / 구현완료로 전환합니다.

### 아이디어 상세 (카드 클릭)

상세 다이얼로그에서 아이디어를 깊이 있게 다룰 수 있습니다.

- **기간 설정**: 시작일 / 종료일 지정 (타임라인 뷰에 반영)
- **상태 워크플로우**: 상단 스텝 바에서 클릭하여 상태 전환
- **논의 탭**: 댓글 작성 / 수정 / 삭제 (본인 댓글만)
- **리소스 탭**: 참고 링크나 파일 URL 첨부 / 삭제
- **발전 경로 탭**: 현재 아이디어에서 파생된 아이디어 목록 확인 및 새로 생성

### 필터 & 정렬

- **검색**: 제목, 내용, 태그로 검색
- **사업체 필터**: 특정 사업부 아이디어만 보기
- **상태 필터**: 특정 상태만 보기 (그리드 뷰에서만)
- **정렬**: 최신순 / 인기순(좋아요) / 논의 많은 순

## 프로젝트 관리

사업부별 프로젝트를 관리하는 페이지입니다.

### 프로젝트 생성

1. **새 프로젝트** 버튼 클릭
2. 사업체 선택 → 프로젝트명 → 설명 → 상태 → 담당자 → 마감일 설정
3. **프로젝트 생성** 클릭

### 프로젝트 카드 기능

각 프로젝트 카드 우측 상단의 **점 세 개** 메뉴:

| 메뉴 | 설명 |
|------|------|
| **수정** | 프로젝트 정보 편집 |
| **보관** | 프로젝트를 보관 처리 (목록에서 숨김) |
| **삭제** | 프로젝트 영구 삭제 (확인 다이얼로그) |

### 진행률

카드 하단의 프로그레스 바는 해당 프로젝트의 아이디어 진행률을 나타냅니다:
- 분자: 승인 + 구현완료 상태인 아이디어 수
- 분모: 전체 아이디어 수

### 탭 필터

상단 탭으로 전체 / HeungHan / Substract Lab / Sensus별 프로젝트를 필터링합니다.

프로젝트 카드를 클릭하면 **캔버스** 페이지로 이동합니다.

## 캔버스

프로젝트별 아이디어를 시각적으로 배치하고 연결하는 화이트보드입니다.

### 기본 조작

| 동작 | 방법 |
|------|------|
| **화면 이동** | 빈 영역 드래그 |
| **확대/축소** | 마우스 휠 또는 좌측 하단 컨트롤 |
| **노드 이동** | 노드를 드래그 |
| **노드 선택** | 클릭 (리사이즈 핸들 표시) |
| **상세 보기** | 아이디어 노드 **더블클릭** |
| **노드 연결** | 한 노드의 핸들(점)에서 다른 노드로 드래그 |
| **노드 삭제** | 노드 선택 후 Delete/Backspace 키 |

### 아이디어 추가

우측 상단 **아이디어** 버튼 → 제목과 내용 입력 → **아이디어 등록**

### 메모 추가

우측 상단 **메모** 버튼 → 색상 선택 (노란색/파란색/초록색/빨간색)

- 메모 안에서 텍스트를 바로 입력/수정할 수 있습니다.
- 메모 우측 상단 X 버튼으로 삭제합니다.
- 선택 후 모서리를 드래그하여 크기를 조절합니다.

### 아이디어 수정

아이디어 노드의 **연필 아이콘**을 클릭하면 수정 다이얼로그가 열립니다.

### 범례

우측 상단 패널에서 상태별 색상을 확인할 수 있습니다:
- 파란색: 제안됨
- 보라색: 논의중
- 초록색: 승인됨

## 알림

### 알림 받기

팀원이 아이디어를 등록하거나 댓글을 남기면 알림이 옵니다.

- **데스크톱**: 사이드바 로고 옆 종 아이콘
- **모바일**: 상단 헤더 우측 종 아이콘

빨간 뱃지 숫자가 읽지 않은 알림 수입니다.

### 알림 관리

- 알림을 클릭하면 해당 페이지로 이동하며 읽음 처리됩니다.
- **모두 읽음** 버튼으로 전체 알림을 한번에 읽음 처리할 수 있습니다.

### 알림 설정

**설정** 페이지에서 알림 종류별로 켜고 끌 수 있습니다:
- 새 아이디어 알림
- 댓글 알림
- 상태 변경 알림

## 마이페이지 / 설정

### 마이페이지 (`/profile`)

프로필 정보를 확인하고 수정할 수 있습니다.

- **이름**: Firebase Auth 프로필명과 연동
- **이메일**: 변경 시 새 이메일로 인증 메일이 발송되며, 인증 후 반영
- **역할**: CEO / CPO / CTO 등
- **자기소개**: 자유롭게 작성

### 설정 (`/settings`)

- 계정 정보 확인 (읽기 전용)
- 알림 설정 토글

## 자주 묻는 질문

**Q. 아이디어를 삭제하려면?**
> 캔버스에서 아이디어 노드를 선택 후 Delete 키를 누르세요. 확인 다이얼로그가 나타납니다.

**Q. 칸반 보드에서 상태를 바꾸려면?**
> 아이디어 카드를 원하는 상태 열로 드래그 앤 드롭하세요.

**Q. 타임라인에 아이디어가 안 보여요.**
> 아이디어 상세에서 시작일/종료일을 설정해야 타임라인에 표시됩니다.

**Q. 다른 사람의 댓글을 수정/삭제할 수 있나요?**
> 아니요. 본인이 작성한 댓글만 수정/삭제할 수 있습니다.

**Q. 로그아웃은 어디서?**
> 사이드바 하단의 **로그아웃** 버튼을 누르세요.
